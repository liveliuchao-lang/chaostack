第1章 推理技术栈分层架构

1.1 从硬件到应用的全栈视图 — 七层精化模型

大模型推理系统可以抽象为七个层次。相对于更粗粒度的六层模型，将"固件与驱动层"从微架构/硬件之间拆分出来，以及将"编译与工具链层"从运行时/微架构之间独立出来，可以更精确地反映 ROCm 软件栈的实际结构——特别是 amdgpu + amdkfd 双内核驱动架构、HSA Runtime、IOMMU/SR-IOV 等服务器部署关键组件在推理系统中扮演的独立角色。

注：本七层模型以 AMD ROCm 为主视角展开，同步标注 NVIDIA CUDA 和国产平台对应组件。不同硬件栈（如 Intel Gaudi 走 SynapseAI、Apple Silicon 走 Metal/MPS）的软件层对应关系不同，不在本表覆盖范围内。

第2章 各层技术栈详细组成与分析

2.1 L2 微架构层 — Tensor Core (NVIDIA) vs WMMA (AMD) 深度对比

这是决定 GPU 推理性能上限的硬件基础。NVIDIA 和 AMD 在矩阵计算硬件上走上了不同的路径，但 2025 年的 RDNA4 WMMA v2 已经大幅缩小了差距。

微架构维度NVIDIA Hopper H100 (SM90)AMD RDNA3 W7900 (gfx1100)AMD RDNA4 R9700 (gfx1201)
矩阵指令集PTX MMA / SASS Warp-Group MMA (WGMMA)WMMA v1 (MFMA) — VGPR 布局复杂, 需跨 lane shuffleWMMA v2 — VGPR 简化, 无数据复制, 无跨 lane shuffle
FP16 矩阵吞吐 (每CU/时钟)1024 ops (mma.sync 16×8×16)512 ops (WMMA v1)1024 ops (WMMA v2 — 2× RDNA3)
INT8 矩阵吞吐 (每CU/时钟)2048 ops512 ops (WMMA v1，借助 dual-issue 可达 ~1024)2048 ops (WMMA v2 — 4× RDNA3 基础速率)
FP8/BF8 原生支持✅ Hopper 原生 FP8 E4M3/E5M2❌ 不支持✅ 原生 FP8/BF8 — 首次在工作站/Pro 级别
4:2 结构化稀疏✅ 有效吞吐 ×2❌ 不支持✅ 支持 — INT4+稀疏 达 1,531 TOPS
TMA (异步张量拷贝)✅ TMA — 独立于 CUDA Core 的硬件单元❌ 无等效硬件 (依赖 LDS+Wavefront 软件调度)❌ 无等效硬件 (但 LDS 扩充至 160KB)
Warp/Wavefront 粒度Warp Size = 32Wavefront Size = 64Wavefront Size = 64 (兼容性关键差异)
Thread Block Cluster✅ SM 间协作 (H100 最大 16 SM)❌ (XCD 间通过 Infinity Fabric)❌ (DCU 间通过 IF)
寄存器管理静态分配 (per-thread 最大 255)静态分配动态寄存器池 — 按需申请/释放, 改善 Occupancy
关键认知： NVIDIA Hopper 的 TMA + Thread Block Cluster + WGMMA 组合在 Attention 等对数据搬运敏感的 kernel 上仍具有架构级优势（FlashAttention 3/4 的 5 路 Warp Specialization 深度依赖 TMA）。AMD RDNA4 的 WMMA v2 通过简化 VGPR 布局和增加动态寄存器管理，在 kernel 开发效率和通用矩阵性能上大幅改善，但缺乏 TMA 等效硬件意味着 Attention kernel 在 RDNA 上的峰值性能仍受限于数据搬运效率。这是 ROCm 版本的 FlashAttention 目前仅支持到 FA2/FA3 (Triton-based)、不支持 FA4 的根因。

8.2 核心结论

推理技术栈 = 纵向贯通七层 × 横向覆盖三平台。

纵向贯通（硬件→固件驱动→编译器工具链→数学计算库→编程运行时→推理引擎→应用编排）决定工程师能"挖多深"。横向覆盖（NVIDIA CUDA + AMD ROCm + 国产加速卡）决定工程师能"走多宽"。

2026 年 AI 芯片格局：NVIDIA 凭借 CUDA 生态的"最后 10%"优势（FA4、TensorRT-LLM、Nsight Profiler）保持高价值场景统治力；AMD ROCm 7.2 + Radeon PRO (特别是 W7900 48GB + R9700 FP8) 在 PyTorch + vLLM 推理的主流场景中已成为生产可用的替代方案；国产平台在国家战略驱动下，在框架移植适配和自主工具链建设方面持续突破。

AMD Radeon PRO 工作站 GPU 在 2026 年的核心价值主张：W7900 (48GB) 是单卡 70B 量化模型的最优 AMD 选择；R9700 ($1,299 · FP8 原生) 是 FP8 推理性价比标杆；R9700S/R9600D 的被动散热设计开启服务器化部署新范式。
